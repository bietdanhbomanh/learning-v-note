Được gọi là toán tử 3 ngôi.
Nhiều trường hợp có thể thay thế cho if ... else và ngắn gọn hơn

syntax:
condition ? ... when true : ... when false;
example:

function tossCoin() {
    var result = Math.random() > 0.5 ? "Sấp" : "Ngửa" ;
    console.log(result);
}

-> Function trên sẽ minh họa việc tossCoin.
